##Tableflip Game Engine

**libraries used:**
* OpenGL / WebGL
* Bullet Physics
* OpenAL
* yaknet
* SOIL / stb_image
* Freetype-gl (binary requires license notice)
* Emscripten
* (linux) X11 / Xi
* (osx) QuartzCore(CoreVideo)
* GLEW
* TinyXML2(for tiled map files)
* Android NDK
